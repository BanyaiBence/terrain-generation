# Bevy Procedural Terrain

A foundational procedural terrain generation setup built with [Bevy 0.18](https://bevyengine.org/) and Rust. 

## Acknowledgements

The project was gladly inspired by the excellent article [Terrain generation in Bevy](https://clynamen.github.io/blog/2021/01/04/terrain_generation_bevy/) by **clynamen**.

## Getting Started

Ensure you have the Rust toolchain installed, then run the project.

```bash
cargo run --release 
```
